<h1>Sensoro Design</h1>

一套开箱即用的 React 组件库。

</div>

## 💻 本地开发

```bash
# 安装依赖
$ pnpm install

# 查看示例
$ pnpm dev

# 启动文档
$ pnpm start
```

## 📚 生态

| 项目 | 介绍 |
| --- | --- |
| [@sensoro-design/charts](/@sensoro-design/charts) | 开箱即用，基于 [G2](https://g2.antv.antgroup.com) 的图表库  |
| [@sensoro-design/plots](/@sensoro-design/plots) | 抹平 G2 与 UI 设计规范的差异，并对 G2 进行扩展  |
